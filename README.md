# machine-learning-clv-project
Machine Learning project to predict Customer Lifetime Value (CLV) using Python, data preprocessing, exploratory data analysis, and regression models.
# Customer Lifetime Value Prediction

## Project Overview
This project predicts Customer Lifetime Value (CLV) using Machine Learning techniques. The notebook includes data preprocessing, exploratory data analysis (EDA), feature engineering, model training, and evaluation to estimate the future value of customers.

## Features
- Data loading and preprocessing
- Handling missing values
- Exploratory Data Analysis (EDA)
- Feature selection
- Machine Learning model training
- Model evaluation
- Prediction of Customer Lifetime Value

## Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Dataset
Customer Lifetime Value (CLV) dataset containing customer demographics, purchase history, and membership information.

## Objective
Build a machine learning model that predicts Customer Lifetime Value to help businesses identify high-value customers and support better marketing and business decisions.
